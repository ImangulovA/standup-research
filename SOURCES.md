# Источники концертов

Откуда мы берём стендап-концерты для корпуса.

## Основной каталог (seed)

- **смешно.xyz — рейтинг полных концертов:** https://smeshno.xyz/youtube/full/
  - Готовый каталог полных стендап-концертов на YouTube с рейтингом по просмотрам.
  - На момент парсинга (2026-07-09): **343 концерта, 196 комиков**.
  - Распарсено в машинный вид: [`sources_smeshno.json`](sources_smeshno.json) (поля: `rank`, `url`, `title`, `year`, `comic`, `views`).
  - Это наш стартовый список: с него начинаем и по нему расширяемся.

## Как расширяемся дальше

Помимо каталога смешно.xyz, полные концерты берём с крупных стендап-каналов и лейблов на YouTube:

- **Stand-Up Club #1** — сольники резидентов
- **Outside Stand Up** — сольные концерты
- **LABELCOM** — сольники (Поперечный, Усович и др.)
- **Standup Store Moscow**
- **Пушка / Pushka**
- **Стендап на ТНТ** (официальный канал) — для мейнстрим-комиков
- **Собственные каналы комиков** — многие выкладывают сольники сами (Данила Поперечный, Саша Долгополов, Иван Усович, Слава Комиссаренко и т.д.)

Принцип отбора концерта:
1. Полный сольный концерт (не нарезки, не выпуски «Открытого микрофона»).
2. Есть русские субтитры (ручные предпочтительнее авто).
3. Указываем исполнителя и год.

## Формат добавления

Один концерт добавляется командой:

```bash
./.venv/bin/python scripts/process.py "<youtube_url>" --artist "Имя Комика" [--year 2023]
```
